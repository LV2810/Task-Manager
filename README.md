## About
Ini adalah sebuah MERN CRUD kecil yang cepat dan sederhana. Task Manager ini dibuat layaknya sebuah to-do list. 
Fitur - fiturnya ada :
- Create Task,
- Edit and Update Task,
- Delete Task.
